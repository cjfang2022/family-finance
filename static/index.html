❓ 如何取得 Gemini API Key？

① 前往 Google AI Studio ⧉

② 點擊左上角的 「Get API Key」 按鈕

③ 如果還沒登入，請登入你的 Google 帳號

④ 點擊 「Create API Key」 產生一組新的金鑰

⑤ 複製整串金鑰文字（以 AIza 開頭），貼到上方的輸入框

⑥ 點擊 「儲存」 即可啟用 AI 自動分類功能
